## Usage

Clone this repo

```bash
git clone https://github.com/miljan-code/keyboardz.git
```

Install necessary dependencies

```bash
npm install
# or
yarn
# or
pnpm install
```

Run the development server

```bash
npm run dev
# or
yarn dev
```
